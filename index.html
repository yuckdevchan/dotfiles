My config files. This includes all my configuration files for my *nix systems and a little website to go with it.

Most of this stuff should work on both GNU/Linux, OpenBSD and FreeBSD although not everything has been thouroughly tested.

How to guide:

Get a hold of the files

To clone: git clone https://github.com/yuckdevchan/dotfiles

If you can't use git then click on the Download ZIP button on this repository or press the button below:

Download

Apply these to your system

Prerequisites

Make sure if you're using a GUI file manager, switch on hidden files visibility. If you're using a terminal then when listing files then use ls -a or lsd -a.

Application

Drag and drop, or CTRL+C and CTRL+V or mv and cp all of the configurations you want on your system. For example, config files and folders in the .config folder, drag the ones you want to your .config file. Do the same with /usr, and any files that are just in the main folder that you downloaded from here, just put the ones you want in your home directory. Be sure to back up any of your pre-existing configuration files as these will be overwritten if you choose to replace them with mine and may not be retrievable.
